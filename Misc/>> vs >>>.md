# 带符号右移与无符号右移

## >>

`>>` 是带符号右移。正数右移高位补 0，负数右移高位补 1。比如：

`4 >> 1`，结果为 2；`-4 >> 1`，结果为 -2。

-4 的补码为：11111100，带符号右移一位得到的补码：11111110，也就是 -2。

### >>>

`>>>` 是无符号右移。无论正数还是负数，高位通通补0。

对于正数而言，`>>` 和 `>>>` 没有区别。

对于负数而言，`-2 >>> 1`，结果是 2147483647（Integer.MAX_VALUE），`-1 >>> 1`，结果是 2147483647（Integer.MAX_VALUE）。

需要注意的是没有无符号左移。
